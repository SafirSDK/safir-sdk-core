DOSE_MEMORY_STATS(1)
====================
:doctype: manpage


NAME
----
dose_memory_stats - simple memory statistics of Safir


SYNOPSIS
--------
*dose_memory_stats*

DESCRIPTION
-----------
Shows the shared memory usage in a Safir system. Updates cyclically.
