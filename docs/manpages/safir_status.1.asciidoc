SAFIR_STATUS(1)
===============
:doctype: manpage


NAME
----
safir_status - Safir system control through Dob objects 


SYNOPSIS
--------
*safir_status*

DESCRIPTION
-----------
safir_status is the service in Safir SDK Core that allows control of the system through Dob objects.
