SAFIR_CONTROL(1)
============
:doctype: manpage


NAME
----
safir_control - Safir system startup and closedown


SYNOPSIS
--------
*safir_control*

DESCRIPTION
-----------
safir_control is the service in Safir SDK Core that controls Safir system startup and closedown.
